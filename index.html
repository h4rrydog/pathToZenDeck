Zen workflow

  • 1. Configure Drupal fields with Fences
  • 2. Build a Zen sub-theme
  • 3. Design your layouts with Zen Grids and Sass

Installation

(1) Zen base theme (Drupal module)

  • Install Zen: drush dl zen
  • Dev version uses SMACSS: drush dl zen-7.x-5.x-dev

(2) Zen Grids (included Sass extension)

  • Install Ruby on Windows using Chocolatey (built into Ubuntu + OS X): cinst ruby
  • Install Compass (Sass dependency): gem install compass

OPTIONAL: Fences (Drupal module)

  • Install Fences: drush dl fences

(3) Create a subtheme

  • Create a custom subtheme: drush cc all drush zen "My First Sub-theme" theme_machine_name
    --description="Our example theme." --without-rtl
  • Enable subtheme + set default in YOUR_DRUPAL_SITE/admin/appearance
